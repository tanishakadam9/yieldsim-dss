# 🌱 YieldSim DSS

### Climate-Driven Crop Yield Simulation & Decision Support System

---

## 📌 Overview

**YieldSim DSS** is a Decision Support System (DSS) designed to analyze climate conditions and simulate their impact on crop yield.

The system helps users understand how environmental factors such as temperature, rainfall, and humidity influence agricultural productivity and supports data-driven farming decisions.

---

## 🎯 Project Objective

* To simulate crop yield based on climate conditions
* To analyze the relationship between climate variables and agricultural output
* To assist farmers and planners in selecting suitable crops
* To provide insights for improving productivity and reducing climate risks

---

## ⚙️ Key Features

* 🌡️ Climate input simulation (Temperature, Rainfall, Humidity)
* 🌾 Crop selection (Wheat, Rice, Maize, etc.)
* 📊 Yield prediction output
* ⚠️ Risk analysis (Low / Moderate / High / Marginal)
* 📈 Interactive dashboard (UI-based DSS)
* 🧠 Decision support recommendations

---

## 🧠 How It Works

1. User inputs climate parameters
2. System processes data using predefined logic/model
3. Simulation engine evaluates conditions
4. Output is generated:

   * Predicted yield
   * Confidence level
   * Risk status
5. System provides recommendations for better decisions

---

## 🧪 Example Scenario

**Input:**

* Temperature: 32°C
* Rainfall: 200 mm
* Humidity: 65%
* Crop: Wheat

**Output:**

* Predicted Yield: 4.08 t/ha
* Confidence: Medium
* Status: Marginal
* Recommendation: Improve irrigation or choose a more suitable crop
